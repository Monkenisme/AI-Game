# ACES: Galaxy Edition

## Over het Project
**ACES: Galaxy Edition** is een intense, top-down "bullet hell" arcade shooter die de grenzen van HTML5 Canvas opzoekt. 
In een universum vol vijandige entiteiten vecht je je door 10 golven van toenemende chaos. Het project combineert procedurele audio, complexe schademodellen voor scheepsonderdelen en een diepgaand upgrade-systeem met elementaire krachten (Fire, Water, Earth, Air).

Dit project is het resultaat van een nauwe samenwerking met Gemini, waarbij we samen de code hebben gefinetuned, bugs hebben platgeslagen en functies zoals de "Secret Boss" en het unieke part-based HP systeem hebben ge√Ømplementeerd.
